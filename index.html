Šį puslapį sukurė Kauno technologijos universiteto, informatikos inžinerijos studijų ketvirto kurso studentas Paulius Alešiūnas. Puslapis skirtas išsamiau parodyti kokias informacines sistemas pavyko sukurti studijų metu. Naudokite menu projektų peržiūrai.

Šie projektai buvo kuriami studijų metu.

Studijų metu įgijau patirties dirbti su:

  • PHP ir HTML/CSS kalbomis leidžiančiomis kurti tinklapio puslapius, kuriuose galima realizuoti aukštą naudotojų duomenų saugumą.
  • Javascript programomis, leidžiančiomis kurti programas naudojančias REST API.
  • Raspberry PI ir Arduino sistemomis, leidžiančiomis kurti realaus laiko sistemas.
  • React Native Expo ir Tailwind.css leidžiančias kurti interaktyvias mobiliąsias programėles.
